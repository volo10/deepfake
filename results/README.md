## Regenerating Results

Run the research notebook to regenerate all figures:

```bash
cd notebooks
jupyter notebook research_analysis.ipynb
```

Or run all cells programmatically:

```bash
jupyter nbconvert --to notebook --execute notebooks/research_analysis.ipynb
```
